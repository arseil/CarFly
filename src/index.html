Вопрос - ответ

Какой размер устройства?
Размер устройства 178*100, диагональ 7 дюймов.
Смогу ли я самостоятельно установить?
Устройство является универсальным, подойдет на большинство автомобилей. В случае некоторых автомобилей, стандартного комплекта проводов будет вполне достаточно.
После установки остались зазоры вокруг устройства, что с этим делать?
В некоторых автомобилях для правильной установки необходима переходная рамка, для того чтобы убрать все зазоры и неровности. Для уточнения обратитесь в техподдержку.
Возможно ли запустить штатный усилитель?
Возможно, для большинства автомобилей предусмотрен Canbus, который приобретается отдельно под модель автомобиля, где присутствует заводской усилитель звука.
Будут ли у меня работать кнопки на руле?
Зачастую для подключения кнопок на руле достаточно набора проводов из комплекта устройства, но бывают случаи, где необходим Canbus. Для уточнения по модели вашего автомобиля, напишите в техподдержку.
